Ticket Analytics Dashboard | Excel, MySQL, Power BI

Description

Developed an interactive Ticket Analytics Dashboard to analyze ticket resolution, revenue, workload distribution, and team performance using Excel, MySQL, and Power BI.

Key Highlights:

Retrieved and validated ticket data using MySQL queries.
Performed data cleaning, transformation, and preparation in Excel and Power Query.
Built an interactive 2-page Power BI dashboard featuring KPIs, trend analysis, forecasting, ticket status, issue categories, workload distribution, and organization-wise insights.
Implemented slicers and date filters for dynamic reporting and drill-down analysis.
Created DAX measures to calculate key metrics such as Total Tickets, Resolved Tickets, Resolution Rate, Revenue, Average Feedback, and team performance indicators.
Designed a user-friendly dashboard to enable quick monitoring of operational performance and support business decision-making.

Tools & Technologies:
Power BI | Power Query | DAX | Excel | MySQL
